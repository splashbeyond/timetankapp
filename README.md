# TimeTank Landing Page

Marketing site for TimeTank, an iOS productivity app by Ephesian28 LLC.

## Routes

- `/` - Landing page
- `/privacy` - Privacy Policy
- `/terms` - Terms of Service

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Verification

```bash
npm run lint
npm run build
```
